### What I learned

During this project I improved my understanding of:

- semantic HTML structure
- CSS spacing systems
- using design tokens with `:root`
- custom font integration
- responsive typography
- hover states and transitions
- box-shadow and border styling

One concept I practiced was creating reusable spacing and typography variables:

```css
:root {
  --spacing-300: 1.5rem;
  --font-size-preset1: clamp(1.25rem, 2vw, 1.5rem);
}
```
